# 文字游戏「三日言」

- **位置**：`/code/scripts/三日无言/`
- **文件**：index.html
- **类型**：文字剧情游戏

## 设定
- 人类得了一种病，会逐渐失去说话和识字的能力，最终退化成动物
- 病程只有三天
- 主角男女可选，有一个姐姐全程照顾

## 游戏机制
- 选项丰富 → 逐渐减少 → 消失
- 界面文字也会缺失
- 最终导向多个结局

## UI 设计
- 邮件风格
- 米白色信封
- 深色顶部条
- 铆钉装饰
- 血迹效果 + 灰尘效果
