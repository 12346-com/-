# 🦌 鹿鸣 — 知识库 & 记忆系统

为 AI 机器人设计的知识管理和对话记忆框架。

---

## 架构

```
┌─────────────────────────────────┐
│         鹿鸣 AI 机器人           │
├─────────────────────────────────┤
│                                 │
│  ┌───────────┐  ┌────────────┐  │
│  │  知识库    │  │  记忆库     │  │
│  │ 长期静态   │  │  对话记录   │  │
│  ├───────────┤  ├────────────┤  │
│  │ 行为准则   │  │ SQLite 存储 │  │
│  │ 用户画像   │  │ 每6h自动总结│  │
│  │ 配置信息   │  │ 分级压缩清理 │  │
│  │ 技术笔记   │  │ 自主搜索回忆 │  │
│  │ 事件日志   │  └────────────┘  │
│  │ 项目进度   │                 │
│  └───────────┘                 │

│  ┌─────────────────────────┐   │
```

## 目录结构

```
鹿鸣的知识库/           # AI 主动维护的长期知识
├── 重要记忆锚点.md      # 入索引入口
├── 行为准则.md          # AI 必须遵守的规则
├── 用户画像.md          # 用户偏好与习惯
├── 技术笔记.md          # 开发备忘
├── 事件日志.md          # 时间线记录
└── 项目进度/            # 各项目独立文档
    ├── 铁锈抽卡.md
    └── 三日言.md

鹿鸣的记忆库/           # 自动运行的对话记忆
├── SELF_GUIDE.md       # AI 自引导说明
├── memory.db           # SQLite 数据库
├── auto_summarize.py   # 每6小时自动总结
├── compress.py         # 分级压缩清理
└── tools/
    └── glm_vision.py   # GLM-4.6V 视觉识别
```

## 特性

- **知识库** — Markdown 多文件结构，职责清晰，AI 主动维护
- **记忆库** — SQLite 单一数据源，无冗余，每6小时自动总结
- **分级压缩** — 重要性决定保留时长（5→30天 / 3→7天 / 1→1天）
- **控制面板** — Web 界面可视化管理记忆和知识库
- **视觉识别** — 集成 GLM-4.6V 图片识别能力

## 技术栈

- Python 3.12
- SQLite
- AstrBot + NapCat
- Playwright
- 高德地图 MCP
